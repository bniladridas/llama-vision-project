Llama Vision Project

An Image Analysis Application by Niladri Das

About the Project

The Llama Vision Project is an innovative application designed for analyzing images. Developed by Niladri Das, it combines a powerful backend with a user-friendly frontend to provide a seamless experience for image processing and visualization.

Technology Stack

  • Backend: Built with FastAPI, a high-performance Python framework, serving as a platform-aware inference server.
  • Frontend: Developed using React, offering a visualization dashboard for user interaction.

Features

The project supports flexible deployment with:

  • Docker: Includes a Dockerfile and docker-compose.yml for hybrid CPU/GPU setups [Dockerfile].
  • Hardware Adaptability: Optimizes for NVIDIA GPUs or Apple Silicon (via Metal) with conditional model loading.
  • Model: Likely integrates the "llama-3.2-90b-vision" model for image reasoning tasks.
